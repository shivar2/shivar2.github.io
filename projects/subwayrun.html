Unity Game Development Project: Endless Runner

Overview:

This project is part of my game development course, created using Unity. The game is similar to the popular "Subway Surfers," where the player character runs, jumps over obstacles, and collects coins. However, in this version, the police character is removed, and the player character gradually loses health over time. In addition to collecting coins, the character can also pick up heart items to restore health. When the character's health is replenished, it begins to dance, adding a fun element to the gameplay.

My Contributions:

  • Designed and developed the entire game independently, from concept to implementation.
  • Implemented various game mechanics such as character animations, obstacle avoidance, and item collection.
  • Created a 2D menu and a 3D explanation bar for the game interface.
  • Incorporated light effects to enhance the visual appeal of the game environment.
  • Developed a system for automatically generating cityscapes within the game world.

Key Features:

  • Endless Runner Mechanics: Players navigate through a continuously generated environment, dodging obstacles and collecting items.
  • Health System: The character's health diminishes over time, with heart items available for recovery.
  • Character Animation: Engaging animations trigger when the character's health is restored.

Performance & Efficiency:

Throughout the development process, I applied various concepts learned during the course, ensuring a well-rounded and functional game. The game was designed to run smoothly on a variety of devices, demonstrating my ability to optimize performance in Unity.

Applications:

  • Game Development: Showcases my skills in game design and programming within Unity.
  • Learning Experience: Reinforced my understanding of game mechanics, user interface design, and animation techniques.
